# Update repo script

Move script to /usr/bin
```
sudo mv updateRepo.sh /usr/bin
``` 

Change permissions
```
sudo chmod a+x /usr/bin/updateRepo.sh
```

Execute:
```
$ updateRepo.sh parameter1...
```
